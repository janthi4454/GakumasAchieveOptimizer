学マスのアチーブメント埋め支援ツールです

## 機能紹介

### 入力ペイン

メイン画面右の入力ペインからアチーブメントの現在値を入力してください

\* が付いているのは「プロデュース回数」です. \
各アイドルの経験値予測をするために必要なので, 1以上の数字を入力してください.

入力ペインの上側にあるアイドルアイコンをクリックすることで, 各アイドルの入力欄にジャンプすることができます. \
入力ペインは `Input` ボタンで閉じることも可能です.

### Settings

第一ステ、第二ステをどれくらい重視するかを決定します. \
ここに入力された割合に応じて, 各レッスンが選ばれると仮定し, 期待経験値を計算します \
総計レッスン回数に影響します

### Snapshots

- `Save`: 現在の入力状況を JSON 形式でダウンロードできます
- `Load`: ファイルに保存した入力状況を画面に反映できます

## 期待経験値算出方法

あるアチーブメントを埋めるのに必要なプロデュース回数をこれまでのプロデュース履歴から算出し, \
そのアチーブを達成したことで得られる経験値を（予測）必要プロデュース回数で割ることで期待経験値量を算出しています.

期待経験値量をアイドル毎に合算して, どのアイドルをプロデュースするのが最も経験値効率が良いか算出します

作者もアチーブメント埋めが終わっていないため, 獲得経験値量が判明していない箇所が多数あるため, \
随時予測を修正予定です
